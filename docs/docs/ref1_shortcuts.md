## 文件
*File*

|命令 |快捷键 |备注 |
| ------------------------------ | ---------------- | ---------- |
|添加新笔记 | Ctrl + N | |
|使用 Templater添加新笔记 | Alt + N |Templater |
|使用 QuickAdd添加新笔记 | Ctrl + Q |QuickAdd |
|打开快速切换器 | Ctrl + O | |
|保存当前打开的文件 | Ctrl + S | |
|关闭当前窗格 | Ctrl + W | |
|将文件移动到其他文件夹 | Alt + M |定制 |
|删除当前笔记 | Ctrl + Shift + D |定制 |
